Photoelectric Effect: Exercises


Exercise 1: Varying the Intensity of the Incident Light

Starting from the default configuration, use the Intensity number box to set the intensity of the incident light to 2. Use the Intensity number box to change the intensity of the incident light to 10. How does this affect the number of photoelectrons ejected per second?

Exercise 2: Varying the Wavelength of the Incident Light

Starting from the default configuration, use the Wavelength number box to set the wavelength of the incident light to 200 nm. (a) Use the Wavelength number box to change the wavelength of the incident light to 400 nm. How does this affect the speed of the ejected photoelectrons? (b) Use the Wavelength number box to change the wavelength of the incident light to 500 nm. Why are no photoelectrons ejected?

Exercise 3: Varying the Voltage Across the Plates

Start from the default configuration. (a) Use the Voltage number box to change the voltage across the plates to 0.59 volts. How is the motion of the photoelectrons affected? (b)Use the Voltage number box to change the voltage across the plates to 0.61 volts. How is the motion of the photoelectrons affected? (c) Use the Wavelength number box to change the wavelength of the incident light to 200 nm, and wait long enough for the original photoelectrons to clear out. Explain why the new photelectrons are able to reach the plate on the right.
