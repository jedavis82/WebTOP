Waves: Exercises


Exercise 1:; Interference of Two In-Phase Point Sources: d = 2λ

Select the overhead point of view. Use the appropriate keyboard entry boxes in the Console to set the parameter values of Point Source 1 to: Amplitude = 6, Wavelength = 8, Phase = 0, X = –50, Y = 8. From the Select Source menu, select Point Source 2. Use the appropriate keyboard entry boxes in the Console to set its parameter values to: Amplitude = 6. Wavelength = 8, Phase = 0, X = –50, Y = –8. Click on the Hide Widgets button.
(a) For observation points far from the source, the intensity of the disturbance is maximum at those angles, call them θm, that fulfill the relation

d sin θm  = mλ,

(1)

where d is the distance between the sources, λ is the wavelength of the waves, and m (an integer) is called the order of the maximum. For the present simulation, find the order, m, and the angular position, qm , of each of the five maxima in the angular interval –90 < q < 90.

(b) For observation points far from the source, the intensity of the disturbance is minimum at those angles, call them qm, that fulfill the relation

d sin θm  = (m + ½) λ,

(2)

where m is an integer. For the present simulation, find the m value and angular position, θm , of each of the four minima in the angular interval –90 < θ < 90.

(c) Click on the animation button on the Console in order to choose the Animation mode. Click on the Play button to see the waves move outwards. Click on the Stop button before proceeding to the next exercise

Exercise 5: Interference of Three In-Phase Point Sources: d = 2λ

In this exercise use an overhead view, and use the appropriate keyboard entry boxes in the Console to set the parameter values of Point Source 1 to: Amplitude = 6, Wavelength = 8, Phase = 0, X = –50, Y = 16. From the Select Source menu, select Point Source 2. Use the appropriate keyboard entry boxes in the Console to set its parameter values to: Amplitude = 6. Wavelength = 8, Phase = 0, X = –50, Y = 0. From the Add Source menu, choose Point, click on the Add button, and use the appropriate keyboard entry boxes in the Console to set the new source’s parameter values to: Amplitude = 6. Wavelength = 8, Phase = 0, X = –50, Y = –16. Click on the Hide Widgets button.

(a) For observation points far from the source, the intensity of the disturbance has  “major” maxima at those angles, call them θm, that fulfill the relation

d sin θm  = mλ,

(5)

where d is the distance between the sources, λ is the wavelength of the waves, and m (an integer) is called the order of the maximum. For the present simulation, find the order and angular position, θm , of each of the three major maxima in the angular interval –90 < θ < 90.

(b) For observation points far from the source, the intensity of the disturbance has “secondary” maxima at those angles, call them θm, that fulfill the relation

d sin θm  = (m + ½) λ,

(6)

where m is an integer. For the present simulation, find the m value and angular position, θm, of each of the four minor maxima in the angular interval –90 < θ < 90.

(c) Choose the animation mode and click on the Play button to see the waves move outwards. Click on the Stop button before proceeding to the next exercise.

Exercise 6: Direction of Propagation of a Linear Wave

Click on the Remove button three times to remove the point sources of the last exercise. Use the Add Source menu to add a linear source whose parameter values are: Amplitude = 4, Wavelength = 15, Phase = 0, and Angle = 0 degrees. Rotate the “water” surface up a little, so that you no longer have an overhead view. Choose the animation mode and click on the Play button to see the waves move. Change the value of the Angle variable to 45 degrees, then 90 degrees, then 135 degrees, … , and then 315 degrees. In each case watch the wave long enough so that you understand the direction the wave is traveling, and what the wavelength of each wave is. Click on the Stop button before proceeding to the next exercise.

Exercise 7: Superposition of Two Linear Waves Traveling in Different Directions

Click on the Remove button to remove the linear wave of the previous exercise. Add a linear wave of Amplitude 4, Wavelength 20, Phase 0, and Angle 45 degrees. Add a linear wave of Amplitude 4, Wavelength 20, Phase 0, and Angle 315 degrees. Rotate the surface so that you have an overhead view.

(a) In what direction will the resultant wave (the superposition of the two linear waves) travel? To verify this, choose the animation mode and click on the Play button.

(b) What is the wavelength of the resultant wave?  Click on the Stop button before proceeding to the next exercise.

Exercise 8: Beats

Click on the Remove button twice to remove the sources of the last exercise.  Add a linear wave whose amplitude is 3.0, wavelength is 6.0, initial phase is 0, and angle is 0. Add a second linear wave that is identical to the first one, except that its wavelength is 6.75. Choose the Animation mode, and click on Play. You should be able to identify traveling wave packets by looking at the edge of the water surface. Click on the Stop button before proceeding to the next exercise.

Exercise 9 One Point Source Plus One Linear Source

(a) Click on the Remove button twice to remove the line sources of the last exercise. Click on the second Viewpoint icon to get an overhead view. Choose the animation mode and click on the Play button.

(b) Click on the Stop button. The pull-down menu in the upper right hand corner of the module allows you to sample the disturbance at a higher or lower resolution than the default value, which is called Medium. Change the resolution to Very Smooth.  Is this interference pattern what you expected? Why is the disturbance for observation positions on the negative x- axis different from that on the positive x-axis?
