Rectangular Aperture: Directions

The default view for the WebTOP window looks like the following:

The WebTOP window has four parts. The top portion is the active WebTOP Scene. The second portion is the X3D Navigation Controls, which allow you to manipulate the scene. The third portion is the Console, which allows you to the change parameter values used in the simulation. The fourth portion is the Recording Panel, which allows you to record and replay WebTOP sessions. You may scroll down to read on, or select the appropriate link from the following items:

How To Change Parameters

The parameters that can be changed are: (a) the wavelength of the incident light, (b) the width of a aperture in the x-direction, (c) the width of a aperture in the x-direction, (d) distance from the aperture plane to the observation plane, and (e) the amount of time that the “film” (the observation screen) is exposed.

Changing the Wavelength of the Incident Light

The wavelength can be changed either by typing the desired value into the Wavelength number box, or by using the cursor to spin the wheel widget that is in the lower left hand corner of the aperture plane. The allowed range of wavelength values is 400 to 700 nm.

Changing the Slit Width in X-Direction

This can be changed by typing the desired value into the X Width number box. It can also be changed by using the cursor to pull horizontally on the blue square widget on the aperture.

Changing the Slit Width in Y-Direction

This can be changed by typing the desired value into the Y Width number box. It can also be changed by using the cursor to pull vertically on the blue square widget on the aperture.

Changing the Distance to the Observation Screen

The distance between the aperture plane and the observation plane can be changed by typing the desired value into the Z Distance number box.

Changing the Exposure Time

The amount of time that the “film” (the observation screen) is exposed can be changed by typing the desired value into the Exposure Time number box.

Using the Controls

The Reset button resets all the parameters to their default values. The Hide Widgets/Show Widgets button hides or shows the widgets in the scene.
